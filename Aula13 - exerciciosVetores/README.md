# Exercícios sobre vetores

## Criar vetores para cada tipo primitivo e exibir o tamanho deles no console

> 	1 - Criar vetores para cada tipo primitivo e exibir o tamanho deles no console;
> 	2 - Criar vetores inicializados para cada tipo primitivo e exibir o tamanho deles no console;
> 	3 - Criar 1 vetor de 5 posições, para que o usuário alimente esses valores;
> 	4 - Criar um vetor com o tamanho informado pelo usuário e depois solicitar que o usuário informe os respectivos valores;
> 	5 - Criar um vetor de inteiros inicializado, para percorrer\n\tDo inicio ao fim\n\tDo fim ao inicio\n\tPara exibir os valore;
> 	6 - Criar um vetor de inteiros não inicializado.
